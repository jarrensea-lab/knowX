## 日志格式

每条发现包含:
- **发现日期**: 何时注意到
- **规律描述**: 发现了什么模式
- **数据支撑**: 有什么数据/观察支持
- **验证状态**: 待验证 / 验证中 / 已确认 / 已失效
- **应用建议**: 如何利用这个规律
