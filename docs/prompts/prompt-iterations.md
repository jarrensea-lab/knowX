# 提示词优化日志

> 手动投喂模板 — 记录提示词的每次修改和效果对比

---

## 日志格式

每条记录请包含：
- **日期**: 修改时间
- **文件**: 涉及的提示词文件/变量名
- **变更类型**: 新增 / 修改 / 删除
- **变更内容**: 具体改了什么
- **原因**: 为什么改
- **效果**: 修改后的实际效果 (好坏都要记录)
- **结论**: 保留 / 回滚 / 继续优化

## 填写示例

```markdown
### 2026-06-01 — 猎手提示词增加止损强调

**文件**: `debate.py` → `HUNTER_PROMPT`
**变更类型**: 修改
**变更内容**: 在猎手提示词的 holdings_advice 中增加了 "必须包含明确的止损价" 要求
**原因**: 猎手输出的止损价经常为空或模糊 ("跌破支撑位")，新手无法执行
**效果**: 
  - 正面: 止损价覆盖率从 60% 提升到 95%
  - 负面: 部分止损价设置过于机械 (固定 -3%)，不区分个股波动率
**结论**: 保留，下次迭代增加波动率自适应止损
```
