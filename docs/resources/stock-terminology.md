# 股票术语表

> 手动投喂模板 — A股交易常用术语中英对照及通俗解释

---

## 基础术语

| 术语 | 英文 | 通俗解释 |
|------|------|----------|
| A股 | A-Share | 在中国境内交易的人民币股票 |
| 上证指数 | Shanghai Composite | 上海交易所的整体表现指标 |
| 深证成指 | Shenzhen Component | 深圳交易所的整体表现指标 |
| 创业板 | ChiNext | 深圳交易所的高科技/成长型板块 |
| 科创板 | STAR Market | 上海交易所的科技创新板块 |
| 北交所 | Beijing Stock Exchange | 北京交易所，服务中小企业 |

---

## 交易术语

| 术语 | 英文 | 通俗解释 |
|------|------|----------|
| 开盘价 | Open | 交易日第一笔成交价 |
| 收盘价 | Close | 交易日最后一笔成交价 |
| 最高价 | High | 当日最高成交价 |
| 最低价 | Low | 当日最低成交价 |
| 成交量 | Volume | 当日成交的股票数量 |
| 成交额 | Turnover | 当日成交的总金额 |
| 换手率 | Turnover Rate | 当日成交股数 / 流通股数 |
| 振幅 | Amplitude | (最高 - 最低) / 昨收，反映波动程度 |
| 量比 | Volume Ratio | 当日每分钟成交 / 前5日平均每分钟成交 |
| 涨跌停 | Limit Up/Down | A股 ±10% 涨跌限制 (北交所 ±30%) |
| T+1 | T+1 Settlement | A股当日买入，次日才能卖出 |
| 做T | T+0 Operation | 利用底仓实现日内买卖（需特殊操作） |

## 技术分析术语

| 术语 | 英文 | 通俗解释 |
|------|------|----------|
| K线 | Candlestick | 展示开盘/收盘/最高/最低价的蜡烛图 |
| 均线 | Moving Average (MA) | 一定时期内的平均价格曲线 |
| 金叉 | Golden Cross | 短期均线上穿长期均线，看涨信号 |
| 死叉 | Death Cross | 短期均线下穿长期均线，看跌信号 |
| MACD | MACD | 趋势跟踪指标，通过均线差值判断方向 |
| RSI | Relative Strength Index | 衡量涨跌力度的指标 (0-100) |
| KDJ | Stochastic | 判断超买超卖的震荡指标 |
| 布林带 | Bollinger Bands | 价格波动的通道 (中轨+上下轨) |
| 支撑位 | Support Level | 价格下跌时可能止跌的位置 |
| 压力位 | Resistance Level | 价格上涨时可能受阻的位置 |
| 突破 | Breakout | 价格有效穿过关键价位 |
| 回踩 | Pullback | 突破后价格回落确认支撑 |

## 基本面术语

| 术语 | 英文 | 通俗解释 |
|------|------|----------|
| PE | Price-to-Earnings | 市盈率 = 股价 / 每股收益，多少年回本 |
| PB | Price-to-Book | 市净率 = 股价 / 每股净资产 |
| ROE | Return on Equity | 净资产收益率，反映赚钱效率 |
| EPS | Earnings Per Share | 每股收益，每持有一股赚多少钱 |
| 总市值 | Market Cap | 公司所有股票的总价值 |
| 流通市值 | Free Float Market Cap | 市场上可自由交易的股票总价值 |
| 分红 | Dividend | 公司把利润分给股东 |
| 除权除息 | Ex-Dividend | 分红后股价自动下调以反映分红金额 |

---

## 资金面术语

| 术语 | 英文 | 通俗解释 |
|------|------|----------|
| 北向资金 | Northbound Capital | 从香港流入A股的外资 (沪股通+深股通) |
| 主力资金 | Institutional Flow | 大额资金的流入/流出 |
| 龙虎榜 | Top Traders List | 每日涨跌幅异常的股票买卖席位 |
| 融资融券 | Margin Trading | 借钱买股(融资)或借股卖出(融券) |
| 大宗交易 | Block Trade | 单笔大额交易，通过专门通道进行 |

---

## 策略术语

| 术语 | 英文 | 通俗解释 |
|------|------|----------|
| 短线 | Short-term Trading | 持股 1-5 天，追求快速收益 |
| 中低频 | Swing Trading | 持股 1-4 周，捕捉波段趋势 |
| 长线 | Long-term Investing | 持股数月到数年，看重基本面 |
| 仓位 | Position Size | 持有的股票金额占总资金的比例 |
| 止损 | Stop Loss | 亏损到一定程度强制卖出，控制风险 |
| 止盈 | Take Profit | 盈利到一定程度卖出，锁定利润 |
| 建仓 | Open Position | 开始买入某只股票 |
| 加仓 | Add Position | 在已有仓位基础上继续买入 |
| 减仓 | Reduce Position | 卖出一部分持仓 |
| 清仓 | Close Position | 全部卖出 |
| 做多 | Go Long | 买入等待上涨 |
| 做空 | Short | 卖出等待下跌 (A股需融券) |
