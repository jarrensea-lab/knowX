# 中低频策略体系 (1-4周)

> 手动投喂模板 — 记录和迭代中低频波段交易策略
>
> 基于项目的「账房」角色分析框架：PE/PB、ROE、MA20/MA60、北向资金、融资余额、行业景气度

---

## 策略框架

### 1. 选股条件

**估值面**:
- [ ] PE(TTM) 在行业 30% 分位以下 (相对低估)
- [ ] PB < 3 (非银行股) 或 PB < 1 (银行股)
- [ ] ROE > 10% (盈利能力筛选)

**趋势面**:
- [ ] MA20 上穿 MA60 (中期金叉)
- [ ] 价格在 MA20 之上
- [ ] MA60 走平或向上 (中期趋势确认)

**资金面**:
- [ ] 北向资金连续 5 日净流入
- [ ] 融资余额稳定或上升 (非恐慌性去杠杆)
- [ ] 机构持仓比例增加

**行业面**:
- [ ] 行业处于景气周期上行阶段
- [ ] 政策面有中长期利好
- [ ] 行业指数在 MA60 之上

### 2. 买入规则

| 条件 | 规则 |
|------|------|
| 买入时机 | 回调至 MA20 附近且缩量企稳 |
| 仓位分配 | 单只不超过总仓位 30% |
| 买入方式 | 分 2-3 批建仓，间隔 2-3 天观察 |

### 3. 卖出规则

| 条件 | 规则 |
|------|------|
| 止盈 | +15% 卖 1/3，+25% 卖 1/3，跌破 MA20 清仓 |
| 止损 | -8% 止损，或跌破 MA60 |
| 基本面止损 | PE 突破历史 90% 分位、ROE 连续两季下滑 |
| 时间止损 | 持有 4 周未达预期 → 减仓评估 |

### 4. 禁止操作

- [ ] PE > 100 的股票 (除非深度研究确认成长性)
- [ ] 业绩预告大幅低于预期的股票
- [ ] 大股东持续减持的股票
- [ ] 行业指数破位下行的板块
