# 短线策略体系 (1-5天)

> 手动投喂模板 — 记录和迭代短线交易策略
>
> 基于项目的「猎手」角色分析框架：MA5/MA10、RSI(14)、量比、资金流、题材热点、龙虎榜

---

## 策略框架

### 1. 选股条件

**技术面**:
- [ ] MA5 上穿 MA10 (金叉)
- [ ] RSI(14) 在 30-70 区间 (避免超买追高)
- [ ] 量比 > 1.5 (放量确认)
- [ ] 价格在 MA5 之上 (短期强势)

**资金面**:
- [ ] 分钟级资金净流入
- [ ] 主力资金连续流入天数
- [ ] 龙虎榜机构席位买入

**题材面**:
- [ ] 所属板块当日涨幅前列
- [ ] 有明确的题材催化剂 (政策/业绩/事件)
- [ ] 板块内有联动效应 (多只同涨)

### 2. 买入规则

| 条件 | 规则 |
|------|------|
| 买入时机 | 盘中回踩 MA5 不破 / 放量突破前高 |
| 仓位分配 | 单只不超过总仓位 20% |
| 买入方式 | 分批买入 (先建 1/3，确认后加仓) |

### 3. 卖出规则

| 条件 | 规则 |
|------|------|
| 止盈 | +5% 开始移动止盈，+10% 卖出 1/2 |
| 止损 | -3% 无条件止损，或跌破 MA5 |
| 时间止损 | 持有 5 天未达预期 → 减仓 |

### 4. 禁止操作

- [ ] 开盘 30 分钟内不追涨
- [ ] 尾盘 15 分钟内不开新仓
- [ ] 不买 ST/退市风险股
- [ ] 不在跌停板接飞刀
- [ ] 不因"便宜"而买入 (关注趋势而非价格)
