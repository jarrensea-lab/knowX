# 买卖原则

> 手动投喂模板 — 入场条件与出场纪律

---

## 买入原则

### 永不买入的情况

- [ ] 大盘处于下跌趋势 (MA60 向下)
- [ ] 个股跌停板
- [ ] ST / *ST 股票
- [ ] 上市不足 60 个交易日的新股
- [ ] 有重大负面消息未消化 (立案调查、财务造假、大股东爆仓)
- [ ] 盘中急拉超过 5% 不追 (大概率冲高回落)

### 买入条件检查清单

**短线买入 (1-5天)**:
- [ ] 大盘环境: MA5 向上，量能正常
- [ ] 个股技术面: MA5 > MA10, 量比 > 1.0
- [ ] 板块效应: 所属板块涨幅前 10
- [ ] 资金面: 近 3 日主力资金净流入
- [ ] 止损位: 已设定明确止损价
- [ ] 仓位: 不超过总仓位 20%

**中低频买入 (1-4周)**:
- [ ] 大盘环境: MA20 走平或向上
- [ ] 估值: PE 处于历史中位数以下
- [ ] 趋势: 价格在 MA60 之上
- [ ] 资金面: 北向资金近 5 日净流入
- [ ] 行业: 行业景气度上行
- [ ] 止损位: 已设定明确止损价 (通常-8%)

---

## 卖出原则

### 必须卖出的情况

- [ ] 触发止损线 (无条件执行)
- [ ] 个股基本面发生重大变化 (业绩变脸、重组失败)
- [ ] 所属行业出现系统性利空
- [ ] 大盘破位 (上证跌破 MA60)
- [ ] 达到目标价 (按计划止盈)

### 考虑卖出的情况

- [ ] 技术面走坏 (跌破关键均线)
- [ ] 量价背离 (价格上涨但量能萎缩)
- [ ] 板块轮动结束 (热点转移)
- [ ] 持有时间到期 (短线 5 天 / 中低频 4 周)

---

## 止损纪律

### 底线 (不可商量)

| 类型 | 止损线 | 说明 |
|------|--------|------|
| 短线 | -3% ~ -5% | 严格执行，不抱幻想 |
| 中低频 | -8% | 可以给更大波动空间 |
| 总账户 | -10% / 月 | 触及后停止交易，复盘反思 |

### 止损执行原则

1. **盘前设定，盘中执行** — 不要临时修改止损价
2. **不移动止损线** — 除非是移动止盈 (只能上移)
3. **不止损的代价** — 小亏变大亏，最终割在最低点

---

## 移动止盈

| 涨幅 | 操作 |
|------|------|
| +5% | 止损线上移至成本价 (保本) |
| +10% | 止损线上移至 +5% |
| +15% | 卖 1/3，止损线上移至 +10% |
| +20% | 卖 1/3，止损线上移至 +15% |
| +25%+ | 考虑清仓 |

---

> 纪律是交易的生命线。策略可以调整，纪律不能妥协。
