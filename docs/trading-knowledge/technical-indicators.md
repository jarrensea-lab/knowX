# 技术指标速查

> 手动投喂模板 — A股常用技术指标的定义、计算和应用

---

## 趋势类指标

### MA (移动平均线)

| 周期 | 用途 | 信号 |
|------|------|------|
| MA5 | 超短线趋势 | 价格在 MA5 之上 → 强势 |
| MA10 | 短线趋势 | 价格跌破 MA10 → 短期转弱 |
| MA20 | 中线趋势 | MA20 方向 = 中期趋势方向 |
| MA60 | 长线趋势 | MA60 之上 → 牛市格局 |
| MA120 | 半年线 | 牛熊分界线之一 |
| MA250 | 年线 | 核心牛熊分界线 |

**金叉/死叉**: 短周期 MA 上穿/下穿 长周期 MA

---

### MACD (指数平滑异同移动平均线)

| 组成 | 说明 |
|------|------|
| DIF | 快线 (12日 EMA - 26日 EMA) |
| DEA | 慢线 (DIF 的 9日 EMA) |
| 柱状图 | DIF - DEA |

**信号**:
- 金叉: DIF 上穿 DEA → 买入
- 死叉: DIF 下穿 DEA → 卖出
- 顶背离: 价格新高, MACD 未新高 → 见顶
- 底背离: 价格新低, MACD 未新低 → 见底

---

## 震荡类指标

### RSI (相对强弱指标)

| 周期 | 常用值 |
|------|--------|
| RSI(6) | 超短线 |
| RSI(14) | 标准短线 (本项目使用) |
| RSI(24) | 中线 |

**信号**:
- RSI > 70: 超买区域 → 考虑卖出
- RSI < 30: 超卖区域 → 考虑买入
- RSI 50 附近: 方向不明

---

### KDJ (随机指标)

| 组成 | 说明 |
|------|------|
| K 值 | 快速确认线 |
| D 值 | 慢速主线 |
| J 值 | 方向敏感线 |

**信号**: K 上穿 D → 金叉买入; J > 100 → 超买; J < 0 → 超卖

---

## 量价类指标

### 量比

```
量比 = 当日每分钟平均成交量 / 过去5日每分钟平均成交量
```

| 量比 | 含义 |
|------|------|
| < 0.5 | 严重缩量 |
| 0.5 - 0.8 | 缩量 |
| 0.8 - 1.5 | 正常 |
| 1.5 - 2.5 | 温和放量 |
| 2.5 - 5 | 明显放量 |
| > 5 | 巨量 (需警惕) |

**项目风控规则**: 量比 > 3 触发风险告警 (放量下跌 = 高风险, 放量上涨 = 中风险)

---

### 换手率

| 换手率 | 含义 |
|--------|------|
| < 1% | 冷门股 |
| 1% - 3% | 正常 |
| 3% - 7% | 相对活跃 |
| 7% - 15% | 高度活跃 |
| 15% - 20% | 异常活跃 |
| > 20% | 极度活跃 (项目风控触发线) |

---

### 振幅

```
振幅 = (最高价 - 最低价) / 昨收价 * 100%
```

| 振幅 | 含义 |
|------|------|
| < 3% | 窄幅震荡 |
| 3% - 7% | 正常波动 |
| 7% - 15% | 大幅波动 |
| > 15% | 剧烈波动 (项目风控触发线) |

---

## 估值类指标

### PE (市盈率)

```
PE = 股价 / 每股收益
PE(TTM): 滚动市盈率 (过去12个月)
PE(静态): 上一个完整财年
```

| PE 范围 | 含义 |
|---------|------|
| < 0 | 亏损 (项目风控触发) |
| 0 - 15 | 低估 |
| 15 - 30 | 合理 |
| 30 - 50 | 偏高 |
| 50 - 100 | 高估 |
| > 100 | 极端 (需深度研究) |
| > 500 | 项目风控触发线 |

**行业差异**: 银行 PE 低 (5-10), 科技 PE 高 (30-50+), 不能跨行业比较 PE。

---

### PB (市净率)

```
PB = 股价 / 每股净资产
```

| PB 范围 | 含义 |
|---------|------|
| < 1 | 破净 (贱卖) |
| 1 - 3 | 合理 |
| > 3 | 偏高 (银行股除外) |

---

### ROE (净资产收益率)

```
ROE = 净利润 / 净资产 * 100%
```

| ROE | 含义 |
|-----|------|
| < 5% | 盈利能力弱 |
| 5% - 10% | 一般 |
| 10% - 15% | 良好 |
| 15% - 20% | 优秀 |
| > 20% | 卓越 |

---

## 项目风控触发线汇总

| 指标 | 触发条件 | 风险等级 | 得分 |
|------|----------|----------|------|
| 价格跌幅 | 亏损 > 7% | High | +0.35 |
| 价格跌幅 | 亏损 > 5% | High | +0.25 |
| 价格跌幅 | 亏损 > 3% | Medium | +0.15 |
| 日内涨跌 | \|涨跌%\| > 9% | High | +0.15 |
| 量比 | > 3 | Medium/High | +0.12 |
| PE | > 500 或 < 0 | Low | +0.05 |
| 换手率 | > 20% | Medium | +0.08 |
| 振幅 | > 15% | High | +0.10 |

> 综合评分 ≥ 0.30 = 高风险，≥ 0.15 = 中风险
