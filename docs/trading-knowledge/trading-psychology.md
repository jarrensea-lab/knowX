## 常见心理偏差

### 1. 损失厌恶 (Loss Aversion)

**表现**: 亏损时不愿止损，希望"等回本再卖"
**后果**: 小亏变大亏，最终割在最低点
**对策**:
- 盘前设好止损价，盘中无条件执行
- 记住: "止损是交易成本，不是亏损"
- 把止损看作"为下一笔交易保存弹药"

### 2. 过度自信 (Overconfidence)

**表现**: 连续盈利后加大仓位、放宽止损、忽视风险
**后果**: 一次大亏吃掉多次小盈
**对策**:
- 盈利后强制降低仓位 (而非提高)
- 连续盈利 3 笔后，暂停一天冷静
- 每笔交易独立看待，不受之前结果影响

### 3. 锚定效应 (Anchoring)

**表现**: 盯着买入价做决策，而非当前市场状态
**后果**: 持有亏损股不放，卖掉盈利股过快
**对策**:
- 忘记成本价，只问: "如果现在是空仓，我愿意在这个价格买入吗？"
- 用移动止盈代替心理锚定

### 4. 从众心理 (Herd Mentality)

**表现**: 看到大家都在买/卖，跟着操作
**后果**: 高位接盘、低位割肉
**对策**:
- 坚持自己的策略体系
- 不因他人赚钱而焦虑 (FOMO)
- "别人贪婪时恐惧，别人恐惧时贪婪"

### 5. 确认偏误 (Confirmation Bias)

**表现**: 只关注支持自己判断的信息，忽略反面信号
**后果**: 持有错误仓位越来越坚定
**对策**:
- 买入前强制列出 3 个反对理由
- 定期检查: "如果我错了，什么信号会告诉我？"

## 交易心态清单

### 开盘前
- [ ] 回顾昨日交易，有无违反纪律
- [ ] 确认今日操作计划 (不临时起意)
- [ ] 检查持仓止损位是否需要调整
- [ ] 心理状态: 冷静、客观、不情绪化

### 盘中
- [ ] 只看计划内的信号，不追逐盘面波动
- [ ] 每笔操作前问: "这是计划内还是冲动？"
- [ ] 盈利不飘，亏损不慌

### 收盘后
- [ ] 记录每笔交易的心理状态
- [ ] 不做盘后诸葛亮 ("早知道就...")
- [ ] 区分 "运气" 和 "能力"
